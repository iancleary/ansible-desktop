* `ansible`,
* `cherrytree`,
* `Visual Studio Code`,
* `Chromium`,
* `docker`,
* `docker-compose`,
* `flameshot`,
* `Flatpak`,
* `Nordvpn`,
* `nodejs`,
* `Peek`,
* `postman`,
* `python3.6`,
* `python3.7`,
* `Signal Desktop`,
* `Slack`,
* `Snap`,
* `Spotify`,
* `Stacer` <https://github.com/oguzhaninan/Stacer>,
* `Standard Notes`,
* `Steam`,
* `Sublime Text`,
* `Telegram`,
* `TickTick`,
* `Tresorit`,
* `yarn`,
* `zsh`

### Configuration

* Configure GNOME keybindings and personal preferences
* Install GNOME themes from [tliron/install-gnome-themes](https://github.com/tliron/install-gnome-themes)
